
---

## ⚙️ Serviços

### 🛒 Product Service
Responsável pelo gerenciamento de produtos.

Funcionalidades:
- Criar produto
- Listar produtos
- Buscar produto por ID

---

### 📦 Order Service
Responsável pelo gerenciamento de pedidos.

Funcionalidades:
- Criar pedidos
- Listar pedidos
- Integração com Product Service

---

## 🛠️ Tecnologias Utilizadas

- Python 🐍
- Flask 
- Docker 🐳
- Docker Compose
- REST APIs

---

### Pré-requisitos
